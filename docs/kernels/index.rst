Kernels
=======

Das Jupyter-Team verwaltet den IPython-Kernel. Zusätzlich zu Python können in
Notebooks auch viele andere Sprachen verwendet werden. Eine Liste der
verfügbaren Kernel findet ihr unter `Jupyter kernels
<https://github.com/jupyter/jupyter/wiki/Jupyter-kernels>`_.
