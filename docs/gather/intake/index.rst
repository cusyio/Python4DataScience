Intake
======

Intake erleichtert das Auffinden, Untersuchen, Laden und Verbreiten von Daten.
Es ist damit nicht nur für Datenwissenschaftler und -ingeniere interessant,
sondern auch für Datenanbieter.

Es läd Daten aus vielen verschiedene Formaten und Typen (s. `Plugin Directory
<https://intake.readthedocs.io/en/latest/plugin-directory.html>`_ und `Intake
Project Dashboard <https://intake.github.io/status/>`_) in übliche
Speicherformate wie Pandas DataFrames, Numpy-Arrays oder Python-Listen.
Anschließend sind sie leicht durchsuchbar und auch für verteilte Systeme
zugänglich. Sollte euch ein Plugin fehlen, könnt ihr auch selbst welche
estellen, wie in `Making Drivers
<https://intake.readthedocs.io/en/latest/making-plugins.html>`_ beschrieben.
