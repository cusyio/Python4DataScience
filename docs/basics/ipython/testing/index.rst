Testen
======

Konzepte
--------

.. glossary::

    Test Case (Testfall)
        testet eine einzelnes Szenario.

    Test Fixture (Prüfvorrichtung)
        ist eine konsistente Testumgebung.

    Test Suite
        ist eine Sammlung mehrerer Test Cases.

    Test Runner
        durchläuft eine Test Suite und stellt die Ergebnisse dar.
