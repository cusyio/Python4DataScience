Git-Tagging
===========

``$ git tag``
    auflisten aller Tags
``$ git tag [name] [commit sha]``
    erstellt einen Tag mit dem Namen ``name`` für den aktuellen Commit

    Mit ``sha`` erhält der spezifische Commit einen Tag, nicht der aktuelle

``$ git tag -a [name] [commit sha]``
    erstellt einen Tag mit dem Namen ``name`` für den aktuellen Commit
