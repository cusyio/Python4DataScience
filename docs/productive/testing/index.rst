     <https://carpentries-incubator.github.io/python-testing/>`_

Concepts
--------

.. glossary::

    Test Case
        tests a single scenario.

        .. seealso::
            * `pytest fixtures
              <https://docs.pytest.org/en/stable/fixture.html>`_

    Test Fixture
        is a consistent test environment.

    Test Suite
        is a collection of several test cases.

    Test Runner
        runs through a test suite and presents the results.
