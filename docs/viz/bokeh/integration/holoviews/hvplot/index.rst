hvPlot
======

Viele Bibliotheken wie :doc:`/viz/matplotlib/pandas/index`, `XArray
<http://xarray.pydata.org/>`_, `Dask <http://dask.pydata.org/>`_, `Streamz
<http://streamz.readthedocs.io/>`_ und `Intake
<http://github.com/ContinuumIO/intake>`_ haben eine High-Level-Plot-API, die
meist auf :doc:`/viz/matplotlib/index` beruhen. Sie bieten jedoch nicht die
Vorteile moderner Web-APIs, wie :doc:`/viz/bokeh/index` und `HoloViews
<http://holoviews.org/>`_.  `hvPlot <https://hvplot.holoviz.org/>`_ ist eine auf
HoloViews basierende High-Level-Plot-API, die eine allgemeine und konsistente
API zum Plotten von allen oben genannten Formaten bietet.

.. seealso::

    * `User Guide <https://hvplot.holoviz.org/user_guide/index.html>`_
    * `Gallery <https://hvplot.holoviz.org/reference/index.html>`_
