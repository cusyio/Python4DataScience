hvPlot
======

`hvPlot <https://hvplot.holoviz.org/>`_ ist eine auf HoloViews basierende
High-Level-Plot-API, die eine allgemeine und konsistente API zum Plotten von
Daten bietet.

    * `User Guide <https://hvplot.holoviz.org/user_guide/index.html>`_
