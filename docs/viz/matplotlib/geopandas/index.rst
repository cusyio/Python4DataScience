GeoPandas
=========

`GeoPandas <http://geopandas.org/>`_ erweitert :doc:`../pandas/index` um
Datentypen, die räumliche Operationen an geometrischen Typen ermöglicht,
wobei die geometrischen Operationen von `Shapely
<https://shapely.readthedocs.io/>`_ ausgeführt werden. Für den Datenzugriff
verwendet GeoPandas `Fiona <https://fiona.readthedocs.io/>`_ und zum Plotten
`descartes <https://pypi.python.org/pypi/descartes>`_ und :doc:`../index`.
