OpenGL
======

Die Open Graphics Library `OpenGL <https://www.opengl.org/>`_ ist eine
API-Spezifikation für 2D- und 3D-Grafikanwendungen. Neben `Vaex
<https://github.com/vaexio/vaex>`_ stehen mit `vispy
<http://vispy.org/>`_ und `glumpy <https://glumpy.github.io/>`_ gleich drei
Python-Bibliotheken zur Verfügung, die OpenGL nutzen.
