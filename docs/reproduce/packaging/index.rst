=========
Packaging
=========

Spack
-----

:doc:`Spack <spack:index>` ist ein flexibler Paketmanager, der mehrere
Versionen, Konfigurationen, Plattformen und Compiler unterstützt, die alle auf
derselben Maschine laufen können.

    :hidden:

    spack/index

Pipenv
------

:term:`pipenv` ist ein Abhängigkeitsmanager für Python-Projekte. Er nutzt
:term:`Pip` zum Installieren von Python-Paketen, er vereinfacht jedoch die
Verwaltung von Abhängigkeiten.
